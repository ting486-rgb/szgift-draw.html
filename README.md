# 🎄 圣诞礼物心愿抽签系统

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-deployed-brightgreen)](https://your-username.github.io/christmas-gift-draw)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个充满节日氛围的在线抽签系统，让团队成员互相抽取心愿，增进感情。

## 🌟 在线体验

**网址：** `https://your-username.github.io/christmas-gift-draw`

*（请将 `your-username` 替换为您的GitHub用户名）*

## ✨ 功能特点

- 🎯 **公平抽签**：每人仅限一次机会，随机抽取其他人的心愿
- 🎁 **神秘惊喜**：不知道会抽到谁的心愿，充满期待
- 💝 **永久记录**：抽签结果永久保存在浏览器本地
- 🎄 **节日氛围**：精美的圣诞主题界面和动画效果
- 👨‍💼 **管理功能**：管理员可查看所有抽签情况和重置系统
- 📱 **响应式设计**：支持手机、平板、电脑访问

## 🎮 使用说明

### 普通用户
1. 访问网站首页
2. 点击"开始抽签"
3. 输入您的英文ID登录
4. 点击"召唤圣诞老人"进行抽签
5. 查看抽到的心愿结果

### 管理员
1. 使用ID `admin` 登录
2. 查看所有用户抽签情况和配对信息
3. 可以重置所有抽签记录
