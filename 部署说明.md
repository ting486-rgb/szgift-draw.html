# 🎁 礼物抽签系统 - 本地部署指南

## 📋 系统文件说明

### 用户端文件
- `personal-draw.html` - **主要用户页面**（发给用户的链接）
- `index-new.html` - 系统首页（可选）

### 管理端文件
- `admin-results.html` - **管理员后台**（查看所有抽签结果）

## 🚀 快速部署方法

### 方法一：使用Python（推荐）
```bash
# 1. 打开终端，进入项目目录
cd /Users/tingtingong/Desktop/drawTool/gift-draw-main

# 2. 启动服务器
python3 -m http.server 8080 --bind 0.0.0.0

# 3. 看到 "Serving HTTP on 0.0.0.0 port 8080" 表示成功
```

### 方法二：使用Node.js
```bash
# 1. 安装 http-server（如果没有）
npm install -g http-server

# 2. 启动服务器
cd /Users/tingtingong/Desktop/drawTool/gift-draw-main
http-server -p 8080 -a 0.0.0.0
```

### 方法三：使用PHP
```bash
cd /Users/tingtingong/Desktop/drawTool/gift-draw-main
php -S 0.0.0.0:8080
```

## ⚠️ 注意事项

1. **网络要求**: 所有用户需要在同一WiFi网络下
2. **防火墙**: 确保8080端口未被阻止
3. **IP地址**: 如果IP变化，需要更新分享链接
4. **数据保存**: 抽签记录保存在用户浏览器本地
5. **服务器**: 保持电脑开机和服务器运行

## 🆘 常见问题

**Q: 用户无法访问页面？**
A: 检查IP地址是否正确，确认在同一网络

**Q: 抽签结果丢失？**
A: 数据保存在浏览器本地，清除浏览器数据会丢失

**Q: 想重新开始抽签？**
A: 在管理后台点击"清空所有记录"

**Q: 服务器意外停止？**
A: 重新运行启动命令即可
